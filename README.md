# 🇬🇧 UK Information Website

A simple and responsive website built with **HTML** that introduces the **United Kingdom**, including its tourist attractions, universities, cities, and the latest news. This project was created to practice HTML structure, navigation, multimedia integration, and webpage layout.

---

## 📖 About the Project

This website provides useful information about the United Kingdom through multiple sections, including:

- 📰 Latest News
- 🏰 Famous Tourist Attractions
- 🌃 Night City Views
- 🎓 Universities
- 📺 Embedded YouTube Video
- 🏙️ Countries and Regions of the UK

The website is designed with a clean layout and is intended as a beginner-friendly web development project.

---

## ✨ Features

- Responsive page structure using HTML5
- Navigation bar with multiple menu items
- High-quality images
- News section
- Tourist attractions section
- University information
- Embedded YouTube video
- Footer with social media icons
- Organized page layout using semantic HTML elements

---

## 🛠️ Technologies Used

- HTML5
